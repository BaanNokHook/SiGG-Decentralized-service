# Readme

## Installation

```bash
$ npm install
```

## Test (using Jest)

```bash

# run eslint and precommit rules
.git/hooks/pre-commit

# all tests
npm run test:all

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```

For end to end tests to work, be sure to use a `.env` file .
